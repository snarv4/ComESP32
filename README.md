# ComESP32

## En nuestro proyecto del ESP32 con WebSockets, ¿Quien actua como Host (Servidor) y quien como Cliente? Define brevemente que responsabilidad tiene cada uno en la comunicacion.

La respuesta: el host seria el ESP32 y el cliente seria el dispositivo que intenta conectarse a el. El host administra las connecciones y procesa informacion, el cliente interactua con el Host para cambiar el estado de algo

## Explica coln tus palabras que representaria la Direccion IP y que representaria el Puerto. ¿Por que es neceasario especificar ambos para que el programa funcione?

La respuesta: la direccion IP es una direccion completa para poder identificar un dispositivo y establecer conecciones con el de manera inalambrica, el pueto es el digito unico de cada dispositivo en una red. es necesario especificar las dos porque asi se logra conectar especificamente con un dispositvo
